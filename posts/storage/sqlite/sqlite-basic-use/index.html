本篇博客记录了 sqlite 的基本使用方法，以备后需。

sqlite 安装

Mac 和 Linux 已经默认安装，使用命令 which sqlite3 即可查到安装目录，一般为：

1

/usr/bin/sqlite3

基本命令

1、创建数据库

1

sqlite3 share.db
