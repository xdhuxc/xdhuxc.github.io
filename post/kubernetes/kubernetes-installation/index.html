使用 kubeadmin 安装 Kubernetes

2018-03-10
技术
约 1506 字 预计阅读 4 分钟 次阅读

文章目录

    • 安装 kubernetes

本篇博客记录了分别使用 kubeadmin 安装 Kubernetes 的详细过程，还记录了安装过程中可能出现的问题及解决方法。

安装 kubernetes

1、修改主机名

1

hostnamectl set-hostname 'k8s-master'

2、禁用 SELinux

1
2

setenforce 0
sed -i --follow-symlinks 's/SELINUX=enforcing/SELINUX=disabled/g' /etc/sysconfig/selinux

3、修改内核参数

1
2
3
4
5
6
7

cat <<EOF >  /etc/sysctl.d/kubernetes.conf
net.bridge.bridge-nf-call-ip6tables = 1
net.bridge.bridge-nf-call-iptables = 1
vm.swappiness=0
EOF

sysctl --system

4、关闭防火墙

1
2

systemctl stop firewalld
systemctl disable firewalld

5、禁用 swap

1

swapoff -a

修改 /etc/fstab 文件，注释掉 Swap 的自动挂载

6、使用如下 kubernetes repo 源 及 docker 源

1
2
3
4
5
6
7
8
9

cat <<EOF > /etc/yum.repos.d/kubernetes.repo
[kubernetes]
name=Kubernetes
baseurl=https://packages.cloud.google.com/yum/repos/kubernetes-el7-x86_64
enabled=1
gpgcheck=1
repo_gpgcheck=1
gpgkey=https://packages.cloud.google.com/yum/doc/yum-key.gpg https://packages.cloud.google.com/yum/doc/rpm-package-key.gpg
EOF

7、安装 docker

1
2
3
4

yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo # 配置 docker repo 源
yum list docker-ce.x86_64  --showduplicates |sort -r # 查看最新的 Docker 版本
yum install docker-ce-18.06
systemctl restart docker && systemctl enable docker

8、安装 dubeadm

1
2

yum install -y kubeadm 
systemctl restart kubelet && systemctl enable kubelet

9、使用 kubeadm 执行命令为：

mkdir -p $HOME/.kube
\cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
chown $(id -u):$(id -g) $HOME/.kube/config
