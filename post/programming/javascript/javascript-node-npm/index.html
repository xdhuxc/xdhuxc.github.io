升级 NPM 和 Node

记录下 npm 和 node 的升级方法。

升级 NodeJS

npm 中有一个模块叫做 n，专门用来做 node.js 版本管理的。

使用如下命令更新 node.js 到最新版本

1
2
3
4

npm install -g n # 安装 n 模块
n stable         # 更新到最新稳定版的 node.js
n latest         # 更新到最新版本的 node.js
n v10.14.2       # 更新到指定版本的 node.js

升级 npm

使用如下命令升级 npm

1

npm -g install npm@next
